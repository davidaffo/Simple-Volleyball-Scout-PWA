# Schema foglio gara

Formato pagina: A4 orizzontale.

## Geometria campi

- Campetti individuali e difesa: mezzo campo 9x9, disegnato quadrato.
- Linea dei 3 metri: a un terzo del mezzo campo dalla rete.
- Cambio palla: campo intero 9x18, con rete al centro e linee dei 3 metri a un terzo di ciascuna meta campo.

## Sezioni

- Intestazione: nome gara e squadra analizzata.
- Dati individuali: una colonna per giocatrice, titolari del primo set prima del resto, escluso il libero.
- Difesa: cinque provenienze di attacco, senza Z5.
- Cambio palla: P1, P6, P5, P4, P3, P2, usando le zone reali degli attacchi registrati.
- Nel cambio palla le traiettorie partono dalla zona d'attacco reale dell'evento: dalla rete per prima linea (Z2/Z3/Z4), dalla linea dei 3 metri per seconda linea (Z1/Z6/Z5).
- Le giocatrici mostrate nel campo del cambio palla sono posizionate dalla zona reale dell'attacco registrato, non ricostruite da ruolo o rotazione teorica.
- Gli attacchi errore (`=`) sono esclusi di default dalle traiettorie del foglio gara; possono essere reinclusi dal filtro del foglio.

## Note

Ogni campetto ha righe note proprie, editabili nell'app e stampate sotto al campo.
